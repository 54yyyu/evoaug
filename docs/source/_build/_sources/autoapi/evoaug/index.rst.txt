:py:mod:`evoaug`
================

.. py:module:: evoaug

.. autoapi-nested-parse::

   EvoAug is a PyTorch package to pretrain sequence-based deep learning models for
   regulatory genomics data with evolution-inspired data augmentations followed by a
   finetuning on the original, unperturbed sequence data.
